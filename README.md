# Banking-Website-Frontend
## Sinha Private Bank 🏦

A modern private banking website with a glassmorphic dark UI, built with pure HTML, CSS, and JavaScript.

## Pages

| Page | Description |
|------|-------------|
| **Home** | Landing page with hero section, stats strip, and feature cards |
| **Dashboard** | Account overview with card, transactions, and spending analysis |
| **Send Money** | Numpad transfer interface supporting UPI, NEFT, RTGS & IMPS |
| **Open Account** | 4-step account onboarding with KYC flow |

## Features

- **Glassmorphic Design** — Frosted glass cards, animated gold orbs, subtle grid texture
- **Sign In Modal** — Login and registration with tab switching
- **Live Numpad** — Interactive amount entry for money transfers
- **Multi-step Form** — Account opening with progress tracking and success state
- **Toast Notifications** — Feedback for transfers, login, and account creation
- **Fully Client-Side** — No frameworks, no dependencies, zero build step

## Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts — Cormorant Garamond + DM Sans + DM Mono

## Getting Started

Just open `sinha_private_bank.html` in any modern browser. No server required.
